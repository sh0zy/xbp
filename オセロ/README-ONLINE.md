# オセロ オンライン

インターネット経由で対戦できるオンラインオセロゲームです。

## セットアップ手順

### 1. 依存パッケージをインストール

```bash
cd オセロ
npm install
```

### 2. サーバーを起動

```bash
npm start
```

サーバーが `http://localhost:3000` で起動します。

### 3. ブラウザでアクセス

```
http://localhost:3000
```

## 遊び方

1. **相手を探す** ボタンをクリック
2. 相手が見つかると自動的にゲーム画面に切り替わります
3. 黒と白が自動的に割り当てられます
4. ターンが来たら石を置いてください
5. ゲーム終了まで続けます

## ゲームルール

- 標準的なオセロ（リバーシ）のルールに従います
- 相手の石を自分の石で挟むと、その石が反転します
- 置ける場所がない場合はパスできます
- 両者がパスするとゲーム終了です

## 技術スタック

- **フロントエンド**: HTML, CSS, JavaScript, Socket.IO Client
- **バックエンド**: Node.js, Express, Socket.IO

## ファイル構成

- `server.js` - バックエンドサーバー
- `index.html` - UI
- `game-online.js` - クライアント側ゲームロジック
- `game.js` - ローカルモード用（参考）
- `style.css` - スタイル
- `package.json` - 依存パッケージ

## ポート変更

デフォルトポートは 3000 です。変更する場合：

```bash
PORT=8080 npm start
```

## ホスティング

Render, Railway, Heroku などで無料ホスティング可能です。Git を使ってデプロイしてください。
